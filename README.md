## dotfiles
Personal Dotfiles with an added install script.

### Stack

- **Hyprland** as the WM
- **Kitty** as the terminal emulator
- **NeoVim** as the texteditor
- **Wofi** a dmenu/rofi-like window switcher and application launcher
- **Starship** prompt
- **zsh** as the shell
